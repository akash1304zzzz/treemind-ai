- **Tool Overview**: A lead generation and CRM management tool built using Emergent, a platform that designs and builds tools based on user prompts. 
- **Key Features**: Finds leads, manages them in a clean CRM, and scores leads based on keyword fit, location, and ICP match. 
- **Setup Process**: User enters target customer, locations, and keywords, then connects Gmail to read Google Alerts. 
- **Functionality**: Agents pull company name, source link, date, and relevant snippet from every alert, providing a searchable dashboard and daily email with top leads. 
- **Data Sources**: Built on Google Alerts and public data, no API needed. 
- **Benefits**: Replaces the need for Apollo or Clay for lead generation, automates CRM management.
